# Projet Infrastructure

## Présentation

Le but de ce projet était de créer une application qui permet de gérer la gestion de jeux vidéos

## Cadre de développement

Ce projet a été créer par [deux étudiants](#auteurs) en première année d'informatique sur une durée de sept demi-journées. La base de donnée devait être sécurisée

## Stack technique 

 - Langage : PHP 
 - Base De Donnée : MariaDB

## Installation

1. Installez [PHP](https://www.php.net/downloads) sur votre ordinateur pour pouvoir lancer le serveur. 
2. Cloner le projet sur votre ordinateur.
3. Lancez le serveur.
